MariaDB can be used to store all parameters exported by ReadMateStatus.
The installation of MariaDB is out of the scope of this project.
Once MariaDB is installed, phpMyAdmin can be used to configure the database.

The steps are straightforward:

1. Create the database manually and add a new database user, if necessary.
2. Use 'mate3.sql' file to auto create tables in your MySQL database (phpMyAdmin is recommended for importing the file).
